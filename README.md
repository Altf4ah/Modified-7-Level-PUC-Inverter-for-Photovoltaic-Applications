# Modified 7-Level PUC Inverter for Photovoltaic Applications

## Overview

This repository presents the design, simulation, and hardware implementation of a **Modified Seven-Level Packed U-Cell (MPUC) Inverter** for photovoltaic applications.

Multilevel inverters are widely used in renewable energy systems to convert DC power generated by solar panels into high-quality AC power suitable for loads or grid integration. Traditional inverter topologies such as Cascaded H-Bridge and Neutral Point Clamped inverters often require a large number of switches and components.

The **Modified Packed U-Cell (MPUC) inverter** provides a compact and efficient alternative by generating multiple voltage levels using fewer components while maintaining high power quality.

This project demonstrates both:

- Simulation validation of the proposed inverter topology
- Hardware implementation of a scaled prototype

The main objectives of this project are:

- Analyse output voltage waveform and harmonic distortion

---

# Inverter Topology

The Modified PUC inverter is a **seven-level multilevel inverter topology** that uses:

- Two asymmetrical DC voltage sources
- Six switching devices
- A specialized switching sequence

The topology produces the following output voltage levels:

```
+Vdc
+2/3 Vdc
+1/3 Vdc
0
-1/3 Vdc
-2/3 Vdc
-Vdc
```

This multilevel output closely approximates a sinusoidal waveform, reducing harmonic distortion and improving power quality.

# Phase 2 — Hardware Implementation

A scaled hardware prototype was developed to validate the inverter performance.

### Hardware Modules

The hardware system consists of three main modules:

### 1. Power Supply Module

- Custom multi-tap transformer
- Bridge rectifier
- Filtering capacitors
- Provides isolated power for gate drivers

### 2. Driver Module

- Optocoupler-based isolation
- Gate driver circuits
- Signal amplification for MOSFET switching

### 3. Inverter Module

- Six MOSFET switches
- Two DC voltage sources
- Converts DC input into multilevel AC output

# Control Strategy

The inverter switching signals are generated using an **Arduino-based control system** implementing a predefined switching table.

This generates the required sequence to produce the **seven-level output waveform**.
